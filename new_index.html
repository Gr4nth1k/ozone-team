Middleware Legend
RequireBuyer
Session must contain an authenticated buyer. Redirects to /login on failure.
RequireVendor
Session must contain an authenticated vendor. Redirects to /vendor/login on failure.
RequireAdmin
Session must have admin role. Returns 403 Forbidden for non-admins.
AuditContext
Global. Injects request ID and user ID into context for audit logging on every request.
